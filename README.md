Day 29 – Password Generator GUI Program
Project Overview

This project is a Password Generator GUI application built with Python. It allows users to generate strong, random passwords easily and copy them for use in accounts.

Features

Generate random passwords of customizable length.

Copy password to clipboard with a single click.

Easy-to-use Graphical User Interface (GUI) using Tkinter.

Designed for security-conscious users who want strong passwords.

Installation

Clone the repository:

git clone https://github.com/JavokhirKuchkanboev/Day-29-Password-Generater-GUI-Program.git


Navigate into the project folder:

cd Day-29-Password-Generater-GUI-Program


(Optional) Create and activate a virtual environment:

python -m venv .venv
# Windows
.\.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate


Install dependencies (if any):

pip install -r requirements.txt

How to Use

Run the program:

python main.py


Click “Generate Password” to create a random password.

Use the “Copy” button to copy it to your clipboard.

Technologies Used
